# Estoque Tech Eletrônicos

Aplicativo mobile Android desenvolvido em React Native com Expo Go para controle de produtos, estoque e vendas de uma empresa fictícia do ramo de eletrônicos.

## Objetivo

Auxiliar pequenos comércios na organização do estoque, evitando vendas de produtos indisponíveis e melhorando o controle de entradas e saídas.

## Funcionalidades

- Cadastro de produtos
- Listagem de estoque
- Controle de quantidade disponível
- Registro de vendas
- Atualização automática do estoque após venda
- Exclusão de produtos
- Resumo com total de produtos, itens em estoque, vendas e faturamento

## Tecnologias utilizadas

- React Native
- Expo Go
- JavaScript
- AsyncStorage

## Como executar

1. Instale o Node.js.
2. Instale o Expo:
```bash
npm install -g expo-cli
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o projeto:
```bash
npm start
```

5. Abra o aplicativo Expo Go no celular e escaneie o QR Code.
